# Code of Conduct

bmg follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).
The short version: be kind, assume good faith, and disagree with
ideas not people.

## Reporting

If someone interacting with this project (in issues, PRs, discussion,
or any other channel) is making the space uncomfortable, please
report it via the GitHub Security Advisory pipeline (so it stays
private) at:

  https://github.com/justinstimatze/be-my-geminis/security/advisories/new

The maintainer will acknowledge within 5 business days and act in
proportion to severity — anything from a private note to a public
ban.

## Scope

This Code of Conduct applies to all bmg project spaces (issues, PRs,
discussion threads, any future chat or community space) and to
public spaces when an individual is representing the project.
